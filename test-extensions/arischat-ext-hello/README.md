# Hello Extension

ArsChat 拡張機能システムの動作確認用サンプル拡張です。

## 機能

### Hello ページ
シンプルな挨拶ページ。拡張機能のページ登録・ナビゲーション連携の動作確認に使用します。

### AI テスト ページ
ArsChat の AI API を呼び出すサンプルページ。`ai:send` / `ai:stream` 権限を使用して、拡張機能からチャットリクエストを送信する方法を示します。

## 権限

| 権限 | 用途 |
|------|------|
| `ai:send` | AI へのメッセージ送信 |
| `ai:stream` | AI レスポンスのストリーミング受信 |

## 開発者向け

```bash
# ビルド
npm run build
```

このサンプルは新しい ArsChat 拡張機能を開発する際のテンプレートとして使用できます。
