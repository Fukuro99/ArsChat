# システムモニター

CPU・メモリ・ディスク使用率をリアルタイムで表示する ArsChat 右パネル拡張機能です。

## 機能

- **CPU 使用率** — コア別・全体の使用率をグラフで表示
- **メモリ使用率** — 使用中 / 空き / 合計メモリを表示
- **ディスク使用率** — 各ドライブの使用状況を表示
- **リアルタイム更新** — 1 秒ごとに自動リフレッシュ

## 使い方

右パネルボタン（画面右上）をクリックするか、右パネルタブから「システムリソース」を選択すると表示されます。

## 権限

この拡張機能は追加権限を必要としません。システム情報の取得は Main プロセスで安全に行われます。

## 技術仕様

| 項目 | 詳細 |
|------|------|
| 更新間隔 | 1000ms |
| 表示形式 | 右パネル（`rightPanel: true`） |
| Main Entry | `main.js` |
| Renderer | `dist/renderer.js` |
